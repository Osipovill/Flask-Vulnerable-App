RCE Уровень - Теория

Удаленное выполнение кода (Remote Code Execution, RCE) происходит, когда злоумышленник может выполнить произвольный код на сервере из-за уязвимости в приложении. Это может привести к полной компрометации системы.

В этом уровне пользовательский ввод используется в системной команде без надлежащей валидации, что позволяет злоумышленнику выполнять произвольные команды на сервере.

Информация об уязвимости

CWE-94: Improper Control of Generation of Code ('Code Injection').

CVE: CVE-2021-33026, CVE-2024-34069
